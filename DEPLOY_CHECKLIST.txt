================================================================================
 TROUBLESHOOTING
================================================================================

Problem: Deployment fails with build error
──────────────────────────────────────────
Solution:
  1. Check "Build Logs" in Vercel dashboard
  2. Look for Python dependency errors
  3. Verify requirements.txt syntax
  4. Try: pip install -r requirements.txt locally

Problem: App returns 500 error after deploying
──────────────────────────────────────────
Solution:
  1. Check "Runtime Logs" in Vercel dashboard
  2. Verify all environment variables are set
  3. Test health endpoint: /health/
  4. Check database connectivity

Problem: Database error or migrations didn't run
──────────────────────────────────────────
Solution:
  1. Check if DATABASE_URL is set
  2. Verify PostgreSQL connection string
  3. Check "Runtime Logs" for migration output
  4. Try manual migration: vercel env pull && python manage.py migrate

Problem: Static files not loading (404 on CSS/JS)
──────────────────────────────────────────
Solution:
  1. Usually resolves on next deployment
  2. Check collectstatic ran in build logs
  3. Try: vercel rebuild

Problem: Admin login doesn't work
──────────────────────────────────────────
Solution:
  Use credentials: admin / admin123 (set in entrypoint.sh)
  If still failing:
  1. Check if migrations ran
  2. Check database connectivity
  3. Try resetting: python manage.py createsuperuser
