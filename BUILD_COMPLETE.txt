================================================================================
QUICK START
================================================================================

1. Run startup script:
   ./runserver.sh

2. In another terminal, start Celery:
   source .venv/bin/activate
   celery -A backend.config worker -l info

3. In another terminal, start Redis:
   redis-server

4. Visit:
   http://localhost:8000/

Default Admin Credentials (after setup):
   Username: admin
   Password: (set during createsuperuser)

================================================================================
KEY WORKFLOWS
================================================================================

RESEARCHER WORKFLOW:
  1. Register at /accounts/register/
  2. Import BFI-44 CSV via /tools/csv-upload/
  3. Run pipeline: /tools/run-pipeline/<volunteer_id>/
  4. View results: /dashboard/volunteer/<volunteer_id>/
  5. Export reports: /dashboard/reports/

PUBLIC USER WORKFLOW:
  1. Visit /live-prediction/
  2. Enter text or Twitter handle
  3. Get instant OCEAN scores
  4. View domain-specific insights

ML PIPELINE FLOW:
  Input Data
    ↓
  Q-Learning Active Selection
    ↓
  BERT Embeddings (768-dim)
    ↓
  GAN Data Augmentation
    ↓
  Lasso Regression (5 traits)
    ↓
  OCEAN Scores + Metrics

================================================================================
NEXT STEPS FOR PRODUCTION
================================================================================

1. Configure PostgreSQL:
   - Set DATABASE_URL in .env.local
   - Run: python manage.py migrate

2. Configure Redis:
   - Ensure Redis server running
   - Set REDIS_URL in .env.local

3. Create superuser:
   - Run: python manage.py createsuperuser

4. Collect static files:
   - Run: python manage.py collectstatic

5. Deploy:
   - Use docker-compose.yml OR
   - Follow DEPLOYMENT.md guide

6. Security:
   - Set DEBUG=False
   - Update ALLOWED_HOSTS
   - Generate new SECRET_KEY
   - Enable HTTPS
