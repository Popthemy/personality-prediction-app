About This Project

This system demonstrates an integrated approach to personality prediction using state-of-the-art machine learning techniques:

  • BERT: Extracts contextual embeddings from social media text (768-dimensional)
  • Q-Learning: Actively selects the most informative posts for training
  • GANs: Augments training data through synthetic sample generation
  • Lasso Regression: Trains sparse, interpretable models for each OCEAN trait
  • BFI-44: Uses validated Big Five Inventory as ground truth for model evaluation
