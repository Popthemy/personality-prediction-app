ML PIPELINE (Strict Order Enforced):
  1. Q-Learning Agent ✓
     - Intelligent post selection based on engagement metrics
     - Saves decisions to Q_LEARNING_LOG model
     - State: engagement features, Action: select/skip

  2. BERT Encoder ✓
     - Processes Q-Learning selected posts only
     - Generates 768-dimensional embeddings
     - Hugging Face bert-base-uncased model
     - Cached for efficiency

  3. GAN Augmenter ✓
     - Generator: 2-layer MLP (768 → 768)
     - Discriminator: 2-layer MLP (768 → 1)
     - Trains on BERT embeddings
     - 20 epochs, 32 batch size
     - Increases training data by 30%

  4. Lasso Regressor ✓
     - 5 separate models (one per OCEAN trait)
     - Input: BERT + GAN synthetic embeddings
     - Output: O, C, E, A, N scores (1-5 scale)
     - Feature importance extraction for interpretability
     - Model serialization to LASSO_MODEL

 QUICK START FOR DEPLOYMENT
================================================================================

1. Environment Setup:
   source venv/bin/activate
   ./runserver.sh

2. Database Setup:
   python manage.py migrate
   python manage.py createsuperuser
   python manage.py collectstatic

3. Start Services:
   # Terminal 1: Django server
   python manage.py runserver

   # Terminal 2: Celery worker
   celery -A backend.config worker -l info

   # Terminal 3: Redis (if not running as service)
   redis-server

   - Landing Page: http://localhost:8000/
